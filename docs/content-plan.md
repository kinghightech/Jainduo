# Jina Path Content Plan

Jina Path is a short-session learning app for Jainism. It now teaches each concept before asking questions: objectives, short teaching cards, an example, misconceptions to avoid, then practice.

## Unit Progression

1. Foundations: what Jainism is, Jina, moksha, diversity of practice, ahimsa.
2. Core Ideas: anekantavada, syadvada, three jewels, standpoint language.
3. Daily Practice: five vows, lay and monastic practice, karma and liberation.
4. Advanced Synthesis: traditions and texts, Digambara and Shvetambara nuance, jiva, ajiva, cosmology, kevala jnana.

## Lesson Loop

1. Objectives: what the learner should understand by the end.
2. Teaching cards: 2-3 short explanations with one key term per card.
3. Model example: a concrete situation showing the concept in use.
4. Misconceptions: what not to confuse with the concept.
5. Practice: questions derived from the teaching cards.
6. Review: relearn completed concepts and retry current weak lessons.

## Exercise Types

- Multiple choice for recognition.
- True and false for quick correction.
- Fill blanks for key terms.
- Select all that apply for ethical scope.
- Matching for term-definition practice.
- Ordering for progression and learning loops.
- Reflection prompts for advanced application.

## Editorial Rules

- Teach shared concepts clearly, then note sectarian and historical nuance where needed.
- Avoid saying every Jain community has identical scriptures or practices.
- Keep the tone educational and non-proselytizing.
- Use scenarios for ethics instead of treating Jainism as only vocabulary.
- Avoid decorative religious imagery unless it directly supports a lesson.

## Sources Consulted

- Encyclopaedia Britannica: Jainism overview, Jain philosophy, Shvetambara, and Digambara entries.
- Harvard Pluralism Project: Jain ethics, three jewels, and five vows overview.
- Jainpedia: three jewels and related concept summaries.
